Commands for GitHUb

git init

git add origin "https:"

Command to pull the files from Remote Repo

1: git pull origin master 

Note: git pull <remote> <branch>

Command to add the file to index 

git add <filename>

git add -A to add all the files to index

Command to commit the file to local repo

git commit -m "Your comments"

git commit -a -m "Comments" (to commit all the files)


Commnads to add the config user

git config --global user.name "hamid"

git config --global user.email 123@gmail.com
